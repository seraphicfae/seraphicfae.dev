## Effortlessly Setup a CobbleVerse Server in Docker

I was inspired to play CobbleVerse after watching [BigPuffer](https://www.youtube.com/watch?v=HunpRbg6k4M) play with Droid.
I looked for templates and found [this one](https://github.com/Blue-Kachina/cobbleverse_server), but it wasn’t put together
very well. I decided to take it and make gradual improvements to it; most notably a better Docker Compose with modpack integrated
right into it. I even added backups and a very simple config file that can be edited via a **.env** file.
