## Setup a simple website with Astro and Tailwind

Boilerplate for whenever I want to create another website.

# How to setup

```bash
git clone https://github.com/seraphicfae/astro-tailwind-starter
cd astro-tailwind-starter
pnpm install
pnpm run dev
```
